CLI 操作

                                    
在调试服务启动后，通过 GDB CLI 与 tool-specific CLI 执行调试命令，并保留原始输出。

                                

                                 空闲
                            

                            

                                

                                    
                                    
                                    
CLI 会话会优先自动探测常见 GDB；若你使用交叉 GDB，可手动填写绝对路径。

                                

                                

                                    
                                    
                                

                                

                                    
                                    
                                

                                

                                    
                                    

                                        
                                        
                                    

                                    
若烧录任务已选择文件，CLI ELF 默认会优先沿用该路径；也可在这里单独上传 ELF / 符号文件。

                                

                                

                                    

                                        
                                        
                                        
                                    

                                    
该卡片会把发送的 CLI 命令和 stdout/stderr 原样写入下方消息框；OpenOCD / pyOCD 独占 monitor 区只在对应 debugger 下显示。
